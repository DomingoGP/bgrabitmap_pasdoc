avifChromaDownsampling = (...);

Uses bilinear filter

Values
  • AVIF_CHROMA_DOWNSAMPLING_AUTOMATIC = 0
  • AVIF_CHROMA_DOWNSAMPLING_FASTEST = 1: Chooses best trade off of speed/quality (same as AVERAGE)
  • AVIF_CHROMA_DOWNSAMPLING_BEST_QUALITY = 2: Chooses speed over quality (same as AVERAGE)
  • AVIF_CHROMA_DOWNSAMPLING_AVERAGE = 3: Chooses the best quality upsampling (same as AVERAGE)
  • AVIF_CHROMA_DOWNSAMPLING_SHARP_YUV = 4: Uses averaging filter

avifChromaUpsampling = (...);
 
Values
  • AVIF_CHROMA_UPSAMPLING_AUTOMATIC = 0
  • AVIF_CHROMA_UPSAMPLING_FASTEST = 1: Chooses best trade off of speed/quality (uses BILINEAR libyuv if available, or falls back to NEAREST libyuv if available, or falls back to BILINEAR built-in)
  • AVIF_CHROMA_UPSAMPLING_BEST_QUALITY = 2: Chooses speed over quality (same as NEAREST)
  • AVIF_CHROMA_UPSAMPLING_NEAREST = 3: Chooses the best quality upsampling, given settings (same as BILINEAR)
  • AVIF_CHROMA_UPSAMPLING_BILINEAR = 4: Uses nearest-neighbor filter
