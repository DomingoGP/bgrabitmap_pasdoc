format: avifRGBFormat;

legal depths [8, 10, 12, 16]. if depth>8, pixels must be UInt16 internally

chromaUpsampling: avifChromaUpsampling;

all channels are always full range

chromaDownsampling: avifChromaDownsampling;

How to upsample from 4:2:0 or 4:2:2 UV when converting to RGB (ignored for 4:4:4 and 4:0:0). Ignored when converting to YUV. Defaults to AVIF_CHROMA_UPSAMPLING_AUTOMATIC.

avoidLibYUV: avifBool;

How to downsample to 4:2:0 or 4:2:2 UV when converting from RGB (ignored for 4:4:4 and 4:0:0). Ignored when converting to RGB. Defaults to AVIF_CHROMA_DOWNSAMPLING_AUTOMATIC.

ignoreAlpha: avifBool;

is available for the avifImage/avifRGBImage combination, then libyuv is used. Default is AVIF_FALSE.

alphaPremultiplied: avifBool;

the alpha bits as if they were all 1.

isFloat: avifBool;

indicates if RGB value is pre-multiplied by alpha. Default: false

pixels: PUInt8;

indicates if RGBA values are in half float (f16) format. Valid only when depth == 16. Default: false
